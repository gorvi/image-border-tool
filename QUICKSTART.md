# 快速入门 ⚡

## 一分钟上手

### 1️⃣ 安装并运行
```bash
# 安装依赖
pip3 install Pillow

# 运行程序
python3 main.py
```

或者使用一键启动脚本：
```bash
./run.sh
```

### 2️⃣ 基础使用流程

```
选择尺寸 → 上传图片 → 添加装饰 → 导出图片
```

## 5分钟完成第一张作品

### 场景1: 制作证件照
1. **左侧面板** 点击 "1寸证件照"
2. **左侧面板** 点击 "📁 上传图片"，选择人像照片
3. **右侧面板** 选择 "白色边框"
4. **顶部工具栏** 点击 "💾 导出图片"
5. ✅ 完成！

### 场景2: 制作装饰海报
1. **左侧面板** 点击 "正方形 1:1"
2. **左侧面板** 点击 "📁 上传图片"，选择背景图
3. **左侧面板** 点击几个喜欢的表情贴纸（❤️⭐😊）
4. 在画布上**拖拽贴纸**调整位置
5. **右侧面板** 选择 "简单边框"
6. **顶部工具栏** 点击 "💾 导出图片"
7. ✅ 完成！

### 场景3: 批量处理相册
1. **左侧面板** 选择尺寸 "正方形 1:1"
2. **左侧面板** 上传一张样例图片
3. 添加贴纸和边框（这将作为模板）
4. **右侧面板** 点击 "📁 批量上传"
5. 按住 `Cmd` 选择多张照片
6. **右侧面板** 点击 "⚡ 批量生成"
7. 选择输出文件夹
8. ✅ 完成！所有图片自动应用相同效果

## 核心功能

| 功能 | 位置 | 说明 |

## 操作技巧

### 贴纸操作
- **添加**: 点击左侧贴纸按钮
- **移动**: 在画布上直接拖拽贴纸
- **删除**: 点击贴纸后，点击工具栏 "🗑️ 删除贴纸"

### 图片适配
- 上传的图片会**自动居中**
- **保持宽高比**不变形
- 自动适配选定的画布尺寸

### 批量处理
- 先用一张图调好效果（贴纸+边框）
- 然后批量上传其他图片
- 所有图片应用**相同的装饰效果**

## 常用快捷键

| 快捷键 | 功能 |
|-------|------|
| `Cmd+Z` (Mac) / `Ctrl+Z` | 撤销（开发中） |
| `Cmd+S` (Mac) / `Ctrl+S` | 导出图片 |

## 输出说明

- **默认格式**: PNG（无损）
- **默认命名**: `tupian_年月日_时分秒.png`
- **图片质量**: 95%（可在代码中调整）
- **批量输出**: 文件名格式 `processed_1_原文件名.png`

## 故障排除

### 问题: 程序无法启动
```bash
# 检查Python版本（需要3.8+）
python3 --version

# 重新安装依赖
pip3 install --upgrade Pillow
```

### 问题: 贴纸不显示
- Mac用户: 系统已内置 Apple Color Emoji 字体，应该正常
- Windows用户: 需要系统有Emoji字体支持

### 问题: 导出的图片在哪里？
- 默认会弹出保存对话框
- 可以选择任意位置保存
- 批量处理需要选择输出文件夹
