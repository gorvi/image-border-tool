# 项目开发规范 (RULES)

## 1. 自动功能规范
### 1.1 自动高亮 (Auto Highlight)
- **默认启用**：`highlight_enabled_var` 必须默认为 `True`。
- **实时生效**：当功能启用时，任何文本输入或变更都必须**立即触发**关键词检测与高亮，严禁要求用户重新勾选才能生效。
- **触发时机**：
  - 文本内容变更时 (Debounced)
  - 样式参数变更时
  - 功能开关启用时

## 2. 交互规范
### 2.1 危险操作确认
- **清除文字**：必须包含 **二次确认** (Two-step confirmation) 流程，防止误操作。
