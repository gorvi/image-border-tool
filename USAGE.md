# 使用指南

## 快速开始

### 1. 安装依赖
```bash
pip3 install -r requirements.txt
```

### 2. 运行程序

**方式一：使用启动脚本（推荐）**
```bash
chmod +x run.sh
./run.sh
```

**方式二：直接运行**
```bash
python3 main.py
```

## 功能说明

### 📏 选择尺寸
- **左侧面板**提供多种预设尺寸：
  - 1寸证件照 (295×413)
  - 2寸证件照 (413×626)
  - 正方形 1:1 (800×800)
  - 横版海报 16:9 (1920×1080)
  - 竖版海报 9:16 (1080×1920)
  - 自定义尺寸

### 📁 上传图片
1. 点击左侧 **"📁 上传图片"** 按钮
2. 选择要处理的图片文件
3. 图片会自动适配到选定的尺寸
4. 支持格式：JPG, PNG, BMP, GIF

### ✂️ 编辑图片
- **重置图片**：恢复到原始上传状态
- 图片会自动居中显示
- 保持原始宽高比

### 😊 添加贴纸
1. 在左侧面板选择喜欢的表情贴纸
2. 点击后贴纸会出现在画布中心
3. **拖拽**贴纸可以移动位置
4. 点击工具栏的 **"🗑️ 删除贴纸"** 可删除选中的贴纸

### 🖼️ 添加边框
- 在右侧面板选择边框样式：
  - 无边框
  - 简单边框 (10px)
  - 粗边框 (20px)
  - 白色边框 (15px)
  - 圆角边框 (10px + 圆角)

### ⚡ 批量处理
1. 点击右侧 **"📁 批量上传"**
2. 选择多张图片（按住 Cmd 多选）
3. 先在主画布上编辑好样式（贴纸、边框等）
4. 点击 **"⚡ 批量生成"**
5. 选择输出目录
6. 程序会将所有图片应用相同样式并导出

### 💾 导出图片
1. 编辑完成后，点击 **"💾 导出图片"**
2. 选择保存位置和文件名
3. 支持导出格式：PNG, JPG

## 快捷键

- `Cmd+Z` (Mac) / `Ctrl+Z` (Windows): 撤销（开发中）
- `Cmd+Shift+Z` (Mac) / `Ctrl+Y` (Windows): 重做（开发中）
- `Cmd+S` (Mac) / `Ctrl+S` (Windows): 导出图片

## 使用技巧

### 1. 制作证件照
```
1. 选择 "1寸证件照" 或 "2寸证件照" 尺寸
2. 上传人像照片
3. （可选）添加白色边框
4. 导出图片
```

### 2. 制作海报
```
1. 选择合适的海报尺寸
2. 上传背景图片
3. 添加装饰贴纸（爱心、星星等）
4. 添加边框增强视觉效果
5. 导出图片
```

### 3. 批量处理相册
```
1. 选择正方形 1:1 尺寸
2. 上传一张样例图片
3. 添加喜欢的贴纸和边框
4. 使用批量上传功能选择所有照片
5. 批量生成，所有照片将应用相同效果
```

## 常见问题

### Q: 贴纸显示为方框？
A: 这可能是系统字体问题。确保系统安装了 Apple Color Emoji 字体（Mac系统默认已有）。

### Q: 图片导出后模糊？
A: 
- 确保原图分辨率足够高
- 选择较大的画布尺寸
- 导出时选择 PNG 格式以保持质量

### Q: 批量处理失败？
A: 
- 检查所有图片格式是否支持
- 确保有足够的磁盘空间
- 查看终端错误信息

### Q: 如何移动贴纸？
A: 直接在画布上点击贴纸并拖拽即可移动位置。

### Q: 可以添加文字吗？
A: 当前版本使用表情贴纸，后续版本会支持自定义文字。
