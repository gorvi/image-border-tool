## 🎨 设计细节

### 微妙的渐变
虽然 Tkinter 限制，通过以下方式模拟：
- 纯白背景
- 细边框分隔
- 浅色系配色
- 统一间距

### 交互反馈
```python
# 悬停效果
白色 → 浅灰 (#F0F0F0)

# 点击效果
浅灰 → 更浅 (#E8E8E8)

# 选中效果
白色 → 蓝色高亮
```

### 视觉层级
```
1级: 标题 (13pt Bold)
2级: 按钮文字 (11-12pt)

```

## 💻 技术实现

### 1. 折叠功能
```python
def toggle_left_panel(self):
    if visible:
        panel.pack_forget()  # 隐藏
    else:
        panel.pack()         # 显示
```

### 2. 悬停效果
```python
btn.bind('<Enter>', lambda e: btn.config(bg=hover_color))
btn.bind('<Leave>', lambda e: btn.config(bg=normal_color))
```
