## 🎯 5分钟快速上手

### 步骤 1: 选择尺寸
左侧面板选择想要的尺寸：
- 证件照（1寸、2寸）
- 社交媒体（正方形、小红书）
- 海报（横版、竖版）

### 步骤 2: 上传图片
点击"📁 上传图片"，选择你的照片

### 步骤 3: 基础编辑
切换到右侧"📐 基础"标签：
- 需要旋转？点击旋转按钮
- 需要翻转？点击翻转按钮

### 步骤 4: 添加装饰
切换到右侧"🎨 装饰"标签：
- 点击贴纸添加（如 ❤️ 😊 ⭐）
- 拖动贴纸调整位置
- 选择边框样式

### 步骤 5: 导出
点击顶部"💾 导出图片"，保存你的作品

## 🎨 场景示例

### 场景 1: 制作证件照
```
1. 选择"1寸证件照"或"2寸证件照"
2. 上传头像照片
3. 选择"白色边框"（如需要）
4. 导出
```

### 场景 2: 小红书图片
```
1. 选择"小红书 4:3"（1242x1656）
2. 上传照片
3. 添加装饰贴纸（如✨、🌸）
4. 选择边框
5. 导出
```

### 场景 3: 生日祝福卡片
```
1. 选择"正方形 1:1"
2. 上传照片
3. 添加贴纸：🎂 🎁 🎈
4. 选择"装饰边框"
5. 导出
```

### 场景 4: 批量处理
```
1. 先用一张图片设计好样式
2. 添加贴纸和边框
3. 切换到"⚡ 批量"标签
4. 点击"批量上传图片"
5. 选择多张图片
6. 点击"批量生成并导出"
7. 一次性生成所有图片
```

## 💡 使用技巧

### 技巧 1: 贴纸调整
- **移动**：点击贴纸后拖动
- **删除**：选中贴纸后点击"🗑️ 删除贴纸"
- **多个贴纸**：可以添加多个，每个都可独立移动

### 技巧 2: 批量处理
- 先用一张样例图片调整好所有效果
- 确认满意后再批量上传其他图片
- 批量处理会应用相同的贴纸位置和边框

### 技巧 3: 撤销与重做
- 任何操作都可以撤销（⌘+Z）
- 支持多步撤销
- 误操作不用担心

### 技巧 4: 自定义尺寸
- 选择"自定义尺寸"
- 手动输入宽度和高度
- 适合特殊需求

## 🐛 遇到问题？

### 问题 1: 程序无法启动
```bash
# 检查Python环境
python3 --version

# 测试模块导入
python3 test_import.py

# 查看详细错误
python3 main.py
```

### 问题 2: Tkinter 崩溃
- 确保使用正确的Python（不是Xcode的Python）
- 使用 `./start.sh` 会自动选择兼容的Python

### 问题 3: 贴纸或边框缺失
```bash
# 重新下载资源
python3 download_assets.py
python3 generate_borders.py
```
